# Splunk SIEM SOC Lab

## Overview
This project demonstrates a hands-on SOC (Security Operations Center) lab using Splunk SIEM for real-time log monitoring, threat detection, and incident investigation.

## Tools Used
- Splunk SIEM
- Log Analysis
- Security Monitoring

## Features
- Real-time log monitoring
- Detection of suspicious activities
- SPL-based threat detection
- Incident investigation workflow

## Use Cases
- Brute force attack detection
- Suspicious login activity
- PowerShell execution monitoring

## Skills Demonstrated
- SIEM Monitoring
- Log Analysis
- Threat Detection
- Incident Response
